# Support

Start with the [setup guide](SETUP.md) and [troubleshooting guide](TROUBLESHOOTING.md).

For reproducible bugs or feature requests, [open a GitHub issue](https://github.com/rschwabco/deck-threads/issues/new). Include:

- macOS version and Mac architecture.
- Stream Deck app version and device model.
- Deck Threads companion and plugin versions.
- Whether Codex Desktop and/or Claude, the companion, and Stream Deck each show as connected.
- Exact steps to reproduce and what you expected.
- Sanitized logs when relevant.

Do not post API keys, credentials, source code, private task titles, project names, full local paths, or screenshots containing confidential work. Replace those values with neutral placeholders before submitting.

Security-sensitive reports should follow [SECURITY.md](../SECURITY.md) instead of a public issue.
